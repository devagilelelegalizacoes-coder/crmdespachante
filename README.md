# CRM DESPACHANTE

Sistema de gestão de orçamentos e fluxos n8n.

## Estrutura do Projeto

- `index.html`: Página principal.
- `assets/`: Arquivos estáticos (JS, CSS, Imagens).
- `dashboard_view.html`: Visualização do dashboard.
- `quotes_view.html`: Gestão de orçamentos.
- `tasks_view.html`: Gestão de tarefas.
- `login.html`: Tela de autenticação.

## Tecnologias

- HTML/CSS/JS (Vanilla)
- Docker (Dockerfile incluído)
- Nginx (nginx.conf incluído)

## Como rodar localmente

Você pode usar o script `iniciar_servidor.bat` ou rodar via Docker.
